# Sportsnet App Scorecard Enhancement

**Date:** 2026-03-03  
**Tags:** #sportsnet #feature-request #scorecards #hockey

## Summary

Add penalty kill information to the expanded scorecard state during power plays.

## Feature Details

When a power play is active and the user expands the scorecard, display penalty kill stats/info for the shorthanded team.

## Current Behavior

- Scorecard shows power play indicator
- Expanded state does not show PK details

## Proposed Behavior

- Expanded scorecard during power play should show:
  - Penalty kill percentage
  - Time remaining on penalty
  - PK unit on ice (if available)
